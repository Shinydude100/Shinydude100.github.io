# Security Policy

## Supported Versions

This repository serves as a live, edge-deployed infrastructure portfolio. Only the main branch is actively supported, monitored, and deployed to the Cloudflare edge.

| Branch / Environment | Supported |
| -------------------- | ------------------ |
| `main` (Live Edge)   | :white_check_mark: |
| Feature Branches     | :x:                |

## Reporting a Vulnerability

As a security-focused infrastructure project, all valid vulnerability reports are taken seriously. 

If you discover a security vulnerability within this repository's codebase, the DNS configuration, or the Cloudflare edge routing, please **do not** open a public issue. 

Instead, please report it directly via [GitHub Private Vulnerability Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) on this repository. 

### Scope of Bounties
*Note: This is a personal infrastructure portfolio. While responsible disclosure is highly appreciated and will be credited, there is no financial bug bounty program attached to this domain.*
