⚡ Bolt: [Performance] Implement async loading for Google Fonts

What: Implemented async loading for Google Fonts by using `media="print" onload="this.media='all'"`.
Why: To eliminate render-blocking resources during initial page load, allowing the browser to render the page structure before fonts are fully downloaded.
Impact: Significantly improves the First Contentful Paint (FCP) metric by preventing Google Fonts from blocking the critical rendering path.
Measurement: Verified via Lighthouse performance checks locally (using `npx htmlhint` for structural integrity and Playwright for visual assurance).
