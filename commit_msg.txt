🎨 Palette: Redesign interactive cards as server dashboard widgets

What: Redesigned `.interactive-card` to match an authentic server dashboard widget using v0 MCP generation.
Why: To upgrade the internal UI with better data visualization (metrics, graphs, LEDs) while preserving the exact text and 3D CSS physics, fulfilling the user's request.
