🎨 Palette: Redesign Terminal UI

What: Redesigned the `.sys-visualizer.terminal-mode` element to mimic a native Windows 11 Command Prompt title bar and terminal execution environment.
Why: To meet the user's request for a raw, native execution environment visual style.
Before/After: The terminal header previously used a simple span with text; now it features a structured Windows 11-style title bar with a left-aligned icon, title, and right-aligned window control buttons (Minimize, Maximize, Close). The terminal body was updated with custom dark slim scrollbars and adjusted typography (Consolas, 14px, tighter line height) for authenticity.
Accessibility: Maintained text contrast and structural semantics. The purely visual SVG controls do not interfere with core screen reading logic.
