## Overview
A hardened, terminal-inspired aesthetic. The UI evokes a secure command-center environment, utilizing a high-contrast dark mode to reduce eye strain during extended low-level system exploration, paired with a distinct alert-yellow for primary telemetry and active states.

## Colors
The palette mimics a secure workspace environment.
- **Primary:** High-visibility yellow. Used strictly for active navigation states, secure requisitions, and primary calls-to-action.
- **Secondary:** Muted slate. Used for secondary interaction nodes to establish a clear visual hierarchy.
- **Neutral:** Clean, highly legible off-white for reading technical metrics and core expertise documentation.
- **Background:** Deep terminal black to allow data arrays and background visual elements to render cleanly without washing out the text.

## Typography
Typography blends technical precision with modern web legibility. Monospace accents are utilized for system outputs, file paths, and secure telemetry nodes to reinforce the infrastructure engineering theme.

## Layout
Spacing and padding follow a strict, grid-based rhythm to maintain the structured, highly organized feel of a system dashboard.

## Shapes
Interactive elements and containers use sharp, tight corner radiuses (4px to 8px) to maintain a precise, utilitarian edge.
