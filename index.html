Windows Maintenance Utility (v1.0)

An all-in-one PowerShell utility designed for IT professionals to streamline essential Windows integrity checks and repairs. This script automates SFC (System File Checker), schedules a comprehensive CHKDSK upon the next boot, and executes DISM Restore Health, generating detailed logs for review.

Key Features:

  • Automated System File Checker (SFC /scannow).
  • Scheduled Disk Error Checking (CHKDSK C: /f /r).
  • Windows Image Repair (DISM RestoreHealth).
  • Automatic logging of all operations.

Requirements & Warnings

This utility requires Administrator privileges to execute successfully. The CHKDSK operation requires a system reboot to complete. Please save all work before running this utility.
