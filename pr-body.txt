🎯 **What:** Implemented async loading for Google Fonts by using `media="print" onload="this.media='all'"`.

💡 **Why:** To eliminate render-blocking resources during initial page load, allowing the browser to render the page structure before fonts are fully downloaded.

✅ **Verification:**
- Verified via `npx htmlhint` for structural integrity.
- Used a Playwright script to verify visual assurance.
- Ignored invalid requests based on constraints (e.g. Cloudflare DNS only, no custom HTTP headers for GH pages, flat structure for DOM).

✨ **Result:** Significantly improves the First Contentful Paint (FCP) metric by preventing Google Fonts from blocking the critical rendering path.
