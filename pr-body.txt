What: Redesigned the visual layout of `.interactive-card` elements to look like authentic server-status dashboard widgets from a secure network interface, incorporating detailed overlays, status bars, and metric grids.
Why: To fulfill the user's request for a more technical, server dashboard aesthetic using the defined design tokens while keeping the 3D hover physics intact.
Before/After: Cards previously had basic styling; they now feature grid overlays, a widget top bar with node IDs and "ONLINE" statuses, a 3-column metric grid (Uptime, Latency, Load), an activity graph (Throughput), and status LEDs in the footer.
Accessibility: All purely decorative visual sub-elements (like grids, scanlines, and dummy metric graphs) were marked with `aria-hidden="true"` to hide them from screen readers, preserving the core text content semantics.
